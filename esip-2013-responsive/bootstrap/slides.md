# Bootstrap

## Responsive Hooks

- desktop: >= 980px
  - `.visible-desktop`, `.hidden-desktop`

- tablet: >= 768px and <= 979px
  - `.visible-tablet`, `.hidden-tablet`

- phone: <= 767px
  - `.visible-phone`, `.hidden-phone`
